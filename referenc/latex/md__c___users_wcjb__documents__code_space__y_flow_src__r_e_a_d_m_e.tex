\item 大端存储与小端存储

\quad{}\quad{}大端存储类似人的正常思维，小端存储机器处理更方便。 \begin{quote}
小端存储（\+Little-\/\+Endian）：数据的高字节存储在高地址中，数据的低字节存储在低地址中 。 \end{quote}
\begin{quote}
大端存储（\+Big-\/\+Endian）：数据的高字节存储在低地址中，数据的低字节存储在高地址中。socket编程中网络字节序一般是大端存储 \end{quote}
